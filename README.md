# Интернет-магазин обуви
Демо-экзамен 09.02.07 Информационные системы и программирование

## Функционал
- Ролевая модель (Гость, Клиент, Менеджер, Администратор)
- Просмотр каталога товаров с фильтрацией и поиском
- Управление заказами
- CRUD для товаров (только администратор)

## Установка
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
